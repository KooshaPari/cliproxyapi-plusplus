# cliproxyapi++

The Plus version of cliproxyapi, adding support for third-party providers on top of the mainline project. Agent-native OpenAI-compatible proxy designed for production and local model routing in the Phenotype ecosystem.

## Features

- **Multi-provider routing** — route requests to OpenAI, Anthropic, local models, and more
- **OpenAI-compatible API** — drop-in replacement for OpenAI SDK clients
- **Agent-native design** — built for agent workflows and multi-model orchestration
- **Production-ready** — designed for reliability in production deployments
